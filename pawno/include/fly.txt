//------------------------------------
В начало мода:
	#include <fly>
//------------------------------------
В public OnPlayerConnect:
	InitFly(playerid);
//------------------------------------
Это к командам:
if(!strcmp(cmdtext,"/fly",true))
{
    StartFly(playerid);
    return 1;
}
if(!strcmp(cmdtext,"/stopfly",true))
{
    StopFly(playerid);
    return 1;
}
//------------------------------------
Как лететь:
Вперед = лететь вперед
Направление полета менять мышкой.
Л. кнопка мышы = повысить высоту
П. кнопка мышы = понизить высоту
бегать = больше скорости
ходить = меньше скорости
